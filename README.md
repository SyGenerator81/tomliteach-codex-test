# TomliTeach GCSE English Worksheet Generator

A simple Chromebook-friendly HTML, CSS and JavaScript tool for generating GCSE English worksheets.

## Files

- `index.html` – page structure and form controls
- `styles.css` – layout, branding and print styling
- `script.js` – worksheet generation logic

## Open on a Chromebook

1. Download or copy the project folder onto your Chromebook.
2. Open the **Files** app and locate the folder.
3. Double-click `index.html` to open it in Chrome.

## Generate a worksheet

1. Select a **Subject Mode**.
2. Enter the **Topic / Question Focus**.
3. Choose a **Target Grade** and **Student Support Level**.
4. Enter the **Lesson Focus**.
5. Click **Generate Worksheet**.

## Save as PDF

1. Click **Print**.
2. In the Chrome print window, choose **Destination → Save to PDF**.
3. Click **Save** and choose your preferred location.

## Move the project folder into Google Drive

1. Open the Chromebook **Files** app.
2. In the left sidebar, select **Google Drive**.
3. Drag the whole project folder into Google Drive.
4. Wait for syncing to complete so the files are available online.
